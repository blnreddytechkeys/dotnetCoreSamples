# .NET Core Samples

This repository contains sample projects demonstrating backend development using **ASP.NET Core and .NET Core**.

## Topics Covered

* ASP.NET Core Web API
* Dependency Injection
* Entity Framework Core
* JWT Authentication
* REST API Development
* Middleware
* Clean Architecture

Technology Stack

C#
ASP.NET Core
SQL Server
Entity Framework Core
Dapper
Purpose

The goal of this repository is to provide learning examples and reference implementations for building scalable backend systems using the .NET ecosystem.
